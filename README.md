A Deep Learning-based web application that classifies uploaded images as either a Dog or a Cat using a Convolutional Neural Network (CNN) built with TensorFlow and deployed using Streamlit.

🚀 Features
Upload images in JPG, JPEG, or PNG format
Real-time image classification
Deep Learning model trained using TensorFlow/Keras
Interactive Streamlit web interface
Automatic model loading from Hugging Face
🛠️ Tech Stack
Python
TensorFlow / Keras
NumPy
Pillow (PIL)
Streamlit
Hugging Face
